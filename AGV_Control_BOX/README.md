# 说明
* **用户可以使用AGV无线呼叫盒呼叫AGV到达指定地点或放行**
* **若将拨码开关全部拨至OFF，上电时进入设置状态，设置无线模块参数**
	* 根据手册设置拨码开关，按下呼叫按钮设置参数，设置成功则蜂鸣器短响
* **否则则为工作状态，按下呼叫按钮，若AGV有应答则亮蓝灯，若超时无应答，则亮红灯，若AGV繁忙，则红蓝灯一起亮**

## 程序需完成的任务

### 底层驱动程序
* 74HC595串入并出的驱动程序
    * 设置74HC595端口值
* 74HC165并入串出的驱动程序
    * 读取74HC165端口值
* 无线模块驱动程序
	* 串口发送接收驱动程序
    * 无线模块参数读取设置
    * 低功耗状态
* ADC数模转换驱动程序
* 看门狗喂狗程序
* 按键扫描

### 应用层程序
* 从串口接收数据中查找指定的字符数组
* 记录应答超时
* 记录亮灯超时
* 记录蜂鸣器超时

## AGV无线呼叫盒状态说明
* **上电时会进行无线模块检测**
    * 若无线模块异常，则蜂鸣器常响，AGV呼叫盒无法使用
* **进入设置状态，蜂鸣器长响**
    * 设置属性成功，蜂鸣器短响
    * 设置属性失败，蜂鸣器长响
    * 若未按照设置表设置，无动作
* **进入工作状态，蜂鸣器短响3声**
    * 状态灯中的蓝灯指示电量，电量越少，则亮的时间越短（满电常亮）
    * 状态灯中的红灯指示充电状态
